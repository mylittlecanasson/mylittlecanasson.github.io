# My Little Canasson - Horse Racing Prediction Application

## Overview

**My Little Canasson** is a full-stack web application frontend built with pure HTML, designed to deliver machine learning-based horse racing predictions. The application serves as a comprehensive interface for users to access AI-generated predictions, historical betting data, and educational content about horse racing strategies.

## Architecture

This repository contains the **frontend presentation layer** of the My Little Canasson application. The frontend is published as static HTML files on GitHub Pages, which are then populated with predictions generated by the backend machine learning system (the `another-canasson` repository).

### Technology Stack

- **Frontend Framework**: Pure HTML (no external frameworks)
- **Styling**: Simple.css CDN (minimal, responsive CSS framework)
- **Visualization**: Chart.js for betting performance charts
- **Hosting**: GitHub Pages (static site)
- **Language**: French (fr-FR)

## Integration with another-canasson

The `another-canasson` repository (backend/ML system) generates:
- Daily horse racing predictions using machine learning algorithms
- Performance data for historical analysis
- Prediction accuracy metrics

**The frontend consumes this data** by:
1. Receiving AI predictions from the backend
2. Formatting predictions into user-friendly HTML
3. Updating historical performance charts
4. Maintaining the appearance and user experience

When the backend publishes new predictions, the HTML files are regenerated with the latest data, and the updated content is deployed to this repository.

  - Color-coded bars

```javascript
function drawBarChart(target_id, target_title, xValues, yValues, barColors) {
    new Chart(target_id, {
        type: "bar",
        data: {
            labels: xValues,
            datasets: [{
                backgroundColor: barColors,
                data: yValues
            }]
        },
        options: {
            legend: {display: false},
            title: {display: true, text: target_title},
            scales: {
                yAxes: [{
                    ticks: {max: 50, min: 0}
                }]
            }
        }
    });
}
```

## Workflow

1. **Prediction Generation**: The `another-canasson` backend ML model generates daily horse racing predictions
2. **HTML Generation**: Predictions are embedded into the index.html template
3. **Historical Data Update**: Betting results and gains are appended to bethistory.html
4. **Publication**: Updated HTML files are committed and pushed to this repository
5. **GitHub Pages Deployment**: Changes automatically published to the live website
6. **User Access**: Users visit the site to view predictions, historical performance, and information
